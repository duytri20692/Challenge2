#HOW TO RUN THE TEST
1. Install Visual Studio.
2. Open the source folder and open 'Challenge2.sln' file. 
3. Show Test Explorer windows: Test > Windows > Test Explorer.
4. On Test Explorer windows, right click on 'TravelInsuranceTest' > Run Selected Tests.
(See 'HowToRunTheTest.png' image if needed)

#VIEW THE REPORT
1. Open the source folder and open 'Test_Execution_Reports' folder.
2. Open 'dashboard.html' file.

Note: In case of any failed test, the test script will take a screenshot and attach it to the report.
Screenshots are placed in 'Defect_Screenshots' folder.
